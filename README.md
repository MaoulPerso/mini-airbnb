# Mini Airbnb 🏠

Un clone simplifié de la plateforme Airbnb développé en **Node.js** avec le framework **Express** et le moteur de template **EJS**.

---

## 🚀 Fonctionnalités
* **Gestion des utilisateurs :** Inscription, connexion et profils (`/users`).
* **Gestion des logements :** Affichage, recherche et détails des annonces (`/logement`).
* **Interface dynamique :** Rendu côté serveur des pages HTML avec **EJS**.
* **Gestion des assets :** Fichiers statiques (CSS, images, JS client) servis via le dossier `public`.

---

## 🛠️ Technologies utilisées
* **Backend :** Node.js, Express
* **Moteur de rendu :** EJS (Embedded JavaScript templates)
* **Middlewares :** Cookie-parser (cookies), Morgan (logs HTTP), Http-errors (gestion des erreurs 404/500).
